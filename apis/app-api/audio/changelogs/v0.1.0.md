# Audio App API Changelog

## v0.1.0 (2026-06-14)

Initial release of the SDKWork Audio App API.

### New Features

#### Speech Synthesis
- `POST /app/v3/api/audio/speech` - Create speech synthesis task
- `GET /app/v3/api/audio/speech/voices` - List available speech voices

#### Audio Transcription
- `POST /app/v3/api/audio/transcriptions` - Create transcription task
- `GET /app/v3/api/audio/transcriptions/{taskId}/segments` - List transcription segments

#### Audio Translation
- `POST /app/v3/api/audio/translations` - Create translation task

#### Sound Effects
- `POST /app/v3/api/audio/sound-effects` - Generate sound effect

#### Task Management
- `GET /app/v3/api/audio/tasks` - List tasks
- `GET /app/v3/api/audio/tasks/{taskId}` - Get task details
- `POST /app/v3/api/audio/tasks/{taskId}/cancel` - Cancel task

#### Real-time Processing
- `POST /app/v3/api/audio/realtime/sessions` - Create real-time session

#### Voice Management
- `GET /app/v3/api/audio/voices` - List voices
- `POST /app/v3/api/audio/voices` - Create voice

#### Workspace Management
- `GET /app/v3/api/audio/workspaces` - List workspaces
- `POST /app/v3/api/audio/workspaces` - Create workspace

#### Export
- `POST /app/v3/api/audio/exports` - Create export task

### Security

- All endpoints require dual-token authentication (Authorization + Access-Token)
- Rate limiting applied per user
- Input validation on all request bodies

### Error Handling

- RFC 7807 Problem Details format for error responses
- Validation errors (400)
- Unauthorized errors (401)
- Rate limit errors (429)

### Pagination

- Cursor-based pagination for list operations
- Configurable page size (default: 20, max: 100)

### Idempotency

- All create operations support idempotency keys
- Keys expire after 24 hours
